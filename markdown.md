# 仁归开源誓约

## 🌿 我们的初心

以「仁归」为念，用技术温暖人心，让每一个需要陪伴的灵魂都能被温柔倾听。我们相信，科技的终极意义是治愈与联结。

## 🤝 开源承诺

1.  **开放共享**：本项目所有代码、设计与文档，均以开源协议发布，任何人可自由使用、学习与改进。
2.  **公益初心**：项目核心功能永久免费，绝不以盈利为目的，所有衍生应用需遵循「治愈优先」的伦理准则。
3.  **尊重边界**：严格保护用户隐私，所有数据仅用于功能优化，绝不泄露、贩卖或用于商业目的。
4.  **协作共建**：欢迎所有认同「仁归」理念的开发者、心理工作者与用户参与共建，共同完善这个温暖的精神家园。
5.  **持续迭代**：我们将持续优化产品体验，让技术更有温度，让陪伴更有力量。

## 📜 伦理底线

- 绝不利用平台传播焦虑、暴力或任何有害内容。
- 绝不诱导用户依赖，始终鼓励用户寻求专业心理帮助。
- 始终将用户的心理健康与安全放在第一位。
